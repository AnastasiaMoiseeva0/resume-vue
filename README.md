# Resume Generator

## Описание
Приложение "Генерация Резюме" — учебный проект на Vue.js. Позволяет редактировать блоки резюме, сохранять и отображать результат. Задание включает разбиение UI на компоненты, работу с emits, валидацию props/emits (дополнительно), динамические компоненты и интеграцию с Firebase.

## Ссылки и ресурсы

- Axios - https://axios-http.com
